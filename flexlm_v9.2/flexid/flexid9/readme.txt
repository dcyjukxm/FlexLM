Installing Aladdin USB Driver Manually
--------------------------------------

(1) Start a command-line windows.
(2) Navigate to c:\program files\globetrotter software\globetrotter flexid drivers\flexid9
(3) type: "hinstall -i" (without quotes)
(4) follow the install instructions.
